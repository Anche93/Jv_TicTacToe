# Java_TicTacToe

Резюме: проект выполнен в рамках учебного задания по основам web-разработки на языке **Java** с использованием фреймворка **Spring**. 
 
## Общая информация
**Проект** - "Крестики-Нолики". \
**Java 18** - язык программирования. \
**Spring Boot 2.7.0** - фреймворк для веб-приложения. \
**Gradle (Kotlin DSL)** - система сборки. \
**ConcurrentHashMap** - потокобезопасное хранилище игр.

### Описание приложения
 - Создание новой игры.
 - Поддерживает множество игр одновременно.
 - Определение первого игрока случайным образом.
 - Принимает ход игрока через REST API.
 - Отвечает ходом компьютера (алгоритм Минимакс).
 - Определяет победу, поражение или ничью.

## Скачать приложение. Запустить приложение
- Перейти в раздел Releases и скачать `TicTacToe-1.0.jar`.
- Запустить приложение двойным щелчком или через терминал выполни команду
`java -jar TicTacToe-1.0.jar`.

### Создать игру
- Через терминал ввести команду `curl -X POST http://localhost:8080/game/create`.
- Или запустить скрипт `./play.sh create`.

Ответ:
```
{
"uuid": "... получен сгенерированный UUID игры ...",
"gameField": {
"gameMatrix": [[0,0,0],[0,0,0],[0,0,0]]
},
"firstPlayer": 1
}
```
`firstPlayer: 1` - первый ход за игроком, `firstPlayer: 2` - первый ход за компьютером.

### Сделать ход
 - Через терминал ввести команду
`curl -X POST http://localhost:8080/game/{полученный UUID} 
-H "Content-Type: application/json" 
-d '{"row": 0, "col": 0}'`.
 - Или запустить скрипт `./play.sh 0 0` (ввести 2 параметра в качества "row" и "col").

### Получить состояние игры
- Через терминал ввести команду `curl http://localhost:8080/game/{полученный UUID}`.
- Или запустить скрипт `./play.sh get`.
